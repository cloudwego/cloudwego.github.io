Why CloudWeGo?

CloudWeGo is an open-source middleware set launched by ByteDance that can be used to quickly build enterprise-class cloud native architectures. The common characteristics of CloudWeGo projects are high performance, high extensibility, high reliability and focus on microservices communication and governance.

{{% /blocks/lead %}} {{< blocks/section color="white" >}} {{% blocks/feature icon="fab" title="Open" %}}
Fully open source, community neutral, compatible with the community open source ecology, pluggable components, CloudWeGo components and other open source components can be integrated or replaced with each other.
{{% /blocks/feature %}} {{% blocks/feature icon="fab" title="Industry-standard" %}}
Contains the components needed to build an enterprise-class cloud-native architecture, allowing users to focus more on business development, meet the current and future needs of user scenarios, and experience the refinement of large-scale scenarios.
{{% /blocks/feature %}} {{% blocks/feature icon="fab" title="Cloud Native" %}}
Quickly build a cloud-native microservice system and develop more reliable, extensible and easy-to-maintain cloud-native applications.

Projects

Kitex

Next generation high performance, highly scalable Golang RPC framework.

Hertz

Next generation high performance, highly scalable Golang HTTP framework.

Netpoll

High-performance NIO (Non-blocking I/O) network library, focusing on RPC scenarios.

Thriftgo

A thrift compiler implemented by Golang supports plug-in mechanism.

Volo

A high-performance and strong-extensibility Rust RPC framework.

Monoio

A thread-per-core Rust runtime with io_uring/epoll/kqueue.

Motore

Async middleware abstraction powered by GAT and TAIT.

Pilota

A thrift and protobuf implementation in pure rust with high performance and extensibility.

Sonic

A blazingly fast JSON serializing & deserializing library, accelerated by JIT and SIMD.

Frugal

A very fast dynamic Thrift serializer & deserializer based on just-in-time compilation.

Fastpb

A faster Protobuf serializer & deserializer.

Cwgo

CloudWeGo All in one Code Generator.

Shmipc

A high-performance IPC solution based on shared memory IPC.

Dynamicgo

Dynamically and efficiently operate data for Go.
