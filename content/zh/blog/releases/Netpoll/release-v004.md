## 优化:

- 默认支持 TCP_NODELAY
- 支持在一个循环中读写
- 返回 nocopy rw 的真实错误
- 更改了循环策略的默认数量
- 重新定义了 EventLoop.Serve arg: Listener -> net.Listener
- 在 DisableGopool 中增加了 API
- 删除了读锁
- 连接 Flush API 调整为阻塞的

## Bug 修复:

- 设置剩余待读取大小。
