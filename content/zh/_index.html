为什么选 CloudWeGo?

CloudWeGo 是一套由字节跳动开源的、可快速构建企业级云原生微服务架构的中间件集合。CloudWeGo 项目共同的特点是高性能、高扩展性、高可靠，专注于微服务通信与治理。

{{% /blocks/lead %}} {{< blocks/section color="white" >}} {{% blocks/feature-wide icon="/img/home/high-perf.png" title="高性能" %}}
整合诸如异步 RPC、流式能力、非阻塞 I/O 网络通信、io_uring、即时编译（JIT）和共享内存通信等先进功能，使得 CloudWeGo 能够在高需求环境下提供卓越性能。
{{% /blocks/feature-wide %}} {{% blocks/feature-wide icon="/img/home/extensibility.png" title="强可扩展性" %}}
采用模块化、分层的框架，提供接口用于功能定制，确保 CloudWeGo 能够以精准和高效的方式满足任何项目的具体需求。
{{% /blocks/feature-wide %}} {{% blocks/feature-wide icon="/img/home/multi-proto.png" title="多协议支持" %}}
提供针对 Thrift、gRPC、HTTP/1、HTTP/2 和 WebSocket 协议的支持，同时具备轻松扩展至其他协议的能力。
{{% /blocks/feature-wide %}} {{% blocks/feature-wide icon="/img/home/tool.png" title="极简化研发流程" %}}
可以提供内置的脚手架代码生成工具和一站式 RPC 调用平台来简化整个研发体系的开发测试流程。

项目

Kitex

下一代高性能、强可扩展的 Golang RPC 框架。

Hertz

下一代高性能、强可扩展的 Golang HTTP 框架。

Netpoll

高性能、I/O非阻塞的网络框架，专注于 RPC 场景。

Volo

高性能、强可扩展的 Rust RPC 框架。

Monoio

基于 io_uring/epoll/kqueue 和 thread-per-core 模型的 Rust Runtime。

Sonic

非常快的由 JIT 和 SIMD 加速的 JSON 序列化和反序列化库。

Shmipc

一个基于共享内存的高性能进程间通讯解决方案。

其它

发现更多 CloudWeGo 开源项目。
