从旧版本 `volo.yml` 迁移到现版本 `volo.yml`

```bash
Usage: volo migrate [OPTIONS]

Options:
  -v, --verbose...  Turn on the verbose mode.
  -h, --help        Print help
  -V, --version     Print version
```

## 如何迁移?

1. 切换到项目根目录(`volo.yml`所在目录) 
2. 执行命令 `volo migrate`
