    文档

    Kitex

    Kitex [kaɪt’eks] 字节跳动开源的 Go 微服务 RPC 框架，具有高性能、强可扩展的特点，在字节内部已广泛使用。如果对微服务性能有要求，又希望定制扩展融入企业内部的治理体系，Kitex 会是一个不错的选择。

    Hertz

    Hertz [həːts] 是字节跳动开源的 Go 微服务 HTTP 框架，具有高易用性、高性能、高扩展性等特点。如果对微服务性能有要求，又希望框架能够充分满足企业内部的可定制化需求，Hertz 会是一个不错的选择。

    Volo

    Volo 是字节跳动服务框架团队研发的轻量级、高性能、可扩展性强、易用性好的 Rust RPC 框架，使用了 Rust 最新的 AFIT 和 RPITIT 特性。

    Netpoll

    Netpoll 是由字节跳动开发的高性能 NIO(Non-blocking I/O) 网络库，专注于 RPC 场景。

    Cwgo

    Cwgo 是 CloudWeGo All in one 代码生成工具，整合了各个组件的优势，以提高开发者的体验。

    Eino 用户手册

    Eino 是基于 Golang 的 AI 应用开发框架
